# Edge-computing-model-split
Simple Reproduction for paper DNN Inference acceleration on the edge, INFOCOM2020, NCIR Intern.
## Requirements
```
apt install docker
apt install docker-compose

python >= 3.8
pytorch >= 1.51
```

## Usage

```shell
cd Edge-computing-model-split/
docker-compse up
```

new terminal
```shell
cd src/
python client.py
```
